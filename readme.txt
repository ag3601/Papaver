一応今の設定

メインヒロインは芥子の魔物のパッチェです（オリキャラ）、ユーザ（パソコン前のあなた）は彼女の歌乞いと伴侶。

ウィーム中央に住んでます。


芥子の魔物は眠りと慰めを司りますが事象を動かせません、終焉と信仰の魔術を使えます。髪と瞳は赤と紫色の多色性。

普段は芥子の花や料理用の芥子の種と薬品を各商会に卸しています、爵位持ちですがまた具体な爵位は決めてません。

取り扱う薬は麻酔薬、鎮痛鎮静剤、咳止め。


イギリスは赤いヒナゲシの花を戦没者の象徴（リメンブランス・デー）にしてますのてアルビクロムにも同じ慣習かあるかもしれません。

植物の系譜の上に司るものはそれなり厄介なのてわざと彼女に喧嘩を売る存在は珍しいかもしれません、気性は割と穏やか。

ちなみに植物の系譜のヒロインにしたのはエマジュリア好きなのて


名前の元ネタはラテン語Requiescat in pace（安らかに眠れ）の教会ラテン語読み

ギリシャ、ローマの文化とそれを継承する西欧文化ではは眠りと死と平穏を象徴するようです


ユーザー以外の相方を入れる場合、相方の名前は芥子のシーのティアにする予定、元ネタはラテン語Retinentia（追憶）。
