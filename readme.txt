里々ゴーストエディタ「さとりすと」 のサンプルゴーストです。
エディタはeditorフォルダに入っています。


使用情報:
このゴーストは、ゴースト作成時のテンプレートとして辞書部分を改変して公開しても構いません。
シェル部分についての権利は作者が異なるため、シェルのreadmeを確認してください。

このゴーストをテンプレートとして改変して公開する場合:
dic_update_supplement.txt にはネットワーク更新先の情報があるので、これは改変を始める前に消しましょう。
間違って作業中に更新をかけてしまったときにデータが上書きされて消えてしまうのを防止できます。
さとりすと本体（editorフォルダ）は外しましょう。ユーザさんが使うことはないですし、無駄に大きいです。
satolist.dllとdic_satolist.txt とsatori_conf.txt内のSAORI設定の「SendSatolist」「FindSatolist」を削除しましょう。
